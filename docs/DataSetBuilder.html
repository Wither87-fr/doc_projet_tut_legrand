    DataSetBuilder

    # new DataSetBuilder(type)

    Crée un constructeur de DataSet, en prenant en compte le type de données

    Pour l'instant ne sont supporté que les type "year" et "month"

    Parameters:
    Name Type Description
    type string

    le type voulu ("year" = données pour un graph sur 12 mois glissant, "month" = données pour un graph sur le mois en cours)
