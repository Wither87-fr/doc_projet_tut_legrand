    ChartConfigBuilder

    Classe permettant de créer unne configuration chartJS de manière plus lisible

    Constructor

    # new ChartConfigBuilder(color, yStartAtZero, yDisplayLines, displayLegend, maintainAspectRatio)

    Instanciation des principaux parametres

    Permet de définir les bases du graph

    Parameters:
    Name Type Default Description
    color string

    la couleur des lignes et des labels
    yStartAtZero boolean

    Si l'ordonnée (Y) doit ou non commencer à 0
    yDisplayLines boolean

    Si les lignes sur l'axe des ordonées doivent être dessinés
    displayLegend boolean true

    Si la légende doit ou non etre affichée (defaut = vrai)
    maintainAspectRatio boolean true

    Si le graphique doit garder ses proportions de base (defaut = vrai)
